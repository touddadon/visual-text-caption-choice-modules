# COCO Caption Selector

Приложение оценивает подписи к изображению и выбирает лучшую.
Картинку можно загрузить свою или взять случайную из набора COCO.
Подписи вводятся вручную — по пять вариантов на картинку.

## Как работает

1. Выбрать картинку: при наличии датасета можно использовать картинки COCO, иначе можно загружать свои
2. Ввести 5 подписей или они заполнятся автоматически если это от COCO
3. Выбрать режим работы модели
4. При нажатии получить лучшую подпись модель на основе метрик выдаёт таблицу с ними и то, какая подпись по её мнению наилучшая

## Режимы оценки

- **Текстовая модель** — cross-encoder на основе roberta-base, оценивает подпись по тексту.
- **Визуальная модель** — CLIP считает соответствие подписи картинке, GroundingDINO проверяет, какие объекты из подписи реально есть на изображении, итог даёт ранкирующая модель.
- **Текст + визуальная модель** — объединяет обе оценки.

## Что нужно рядом с app.py

```text
dataset_df.csv
cross_encoder_scorer.pt
visual_ranknet_model.pt
```

Для режима случайной картинки нужна папка с изображениями COCO рядом с проектом:

```text
train2014/
val2014/
images/
```

Имена файлов в папке должны совпадать с колонкой `file_path` в `dataset_df.csv`.
Если COCO-картинок нет — можно загружать собственную картинку.

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

Открыть в браузере:

```text
http://127.0.0.1:7860
```

Запуск на сервере с доступом извне:

```bash
python app.py --server-name 0.0.0.0 --server-port 7860 --share
```

С флагом `--share` в выводе появится публичная ссылка вида `https://....gradio.live`.

## Запуск из ноутбука

Открыть `run_gradio.ipynb` и выполнить все ячейки.

## Подписи на английском

Модели обучены на англоязычных подписях COCO, поэтому стоит работать на английском языке.
